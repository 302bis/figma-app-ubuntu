# Figma, Snap Package (unofficial)
This is a Snap Package of the Awesome Design Tool called **Figma**.

This package, its a web wrapper of the [Figma-web-app](https://figma.com) packed using [Electron](http://electron.atom.io). It lets you install Figma on every distro who accept Snap and easily access it from system men√∫s or the Ubuntu Dash, instead of your web browser.

## Install
sudo snap install --edge figma
